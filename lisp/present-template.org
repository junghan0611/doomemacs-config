# org-tree-slide 발표 템플릿
# 사용법:
#   1. 이 파일을 복사하여 발표 내용 작성
#   2. SPC j p p → 발표 시작
#   3. C-→ / C-← → 슬라이드 이동
#   4. SPC j p q → 발표 종료
#
# 규칙:
#   - * 레벨 1 = 슬라이드 (org-tree-slide이 네비게이트)
#   - ** 레벨 2 = 슬라이드 내 소제목
#   - COMMENT 키워드 = 발표자 노트 (슬라이드에서 안 보임)
#   - 표, 코드블록, 인용 모두 그대로 렌더링

* COMMENT 발표자 노트

여기는 발표 중에 보이지 않는 메모 공간.
org-tree-slide이 COMMENT 헤딩을 건너뛴다.

- 예상 질문 1: ...
- 예상 질문 2: ...
- 시간 배분: 전반부 10분, 후반부 15분, Q&A 5분
